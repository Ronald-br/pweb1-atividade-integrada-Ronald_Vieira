# 📌 Descrição do Projeto

Este projeto consiste em um sistema web desenvolvido em Laravel com o objetivo de praticar:

- Rotas básicas e com parâmetros
- Views Blade
- Controllers
- Fluxo MVC
- Formulários GET e POST
- Resource Controller
- Versionamento com Git e GitHub

O sistema simula um site institucional fictício chamado **TechEdu**.
